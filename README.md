# mcbird-chick

## chick, sample of egg ver. 1.5.2.

[English](README.md) / [日本語](README-jp.md)

### Usage

mcbird-chick is not enabled by default.

After loading it alongside mcbird-egg, enable it with the following command:

```mcfunction
/trigger egg set 101
```

trigger egg is the trigger functionality possessed by egg.

```mcfunction
/trigger egg
```

Executing this displays help for the command corresponding to the number.

To obtain the toolkit:

```mcfunction
/trigger egg 102
```

Execute this command.

You can obtain a batch of nearly pre-generated dedicated items and enchantment books.

Combining dedicated items and enchantment books at an anvil enables special right-click actions.

Dedicated items can also be generated by combining weapons and lapis lazuli at a smithing table.
