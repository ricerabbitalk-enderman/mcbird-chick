# mcbird-chick

## chick, sample of egg ver. 1.5.x.

[English](README.md) / [日本語](README-jp.md)

### 使い方

mcbird-chick はデフォルトで有効化されません。

mcbird-egg と共に読み込んだ後、以下のコマンドで有効化します。

```mcfunction
/trigger egg set 101
```

trigger egg は egg が持つトリガー機能です。

```mcfunction
/trigger egg
```

これを実行することで、番号に対応したコマンドのヘルプが表示されます。

ツールキットを取得する場合は

```mcfunction
/trigger egg 102
```

このコマンドを実行してください。

ほぼ生成済みの専用アイテム・エンチャントの本を一括で取得できます。

専用アイテムとエンチャントの本を金床で合成することで右クリックで特殊な動作をします。

専用アイテムは鍛冶台で武器とラピスラズリを合成することでも生成可能です。
